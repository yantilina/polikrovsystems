# Polikrovsystems

Корпоративный сайт компании по устройству и ремонту мягкой кровли (г. Салават).

## 🌐 Live Demo
https://pks0102.ru/

## 📌 О проекте
Разработка сайта для реального клиента с нуля: от верстки до размещения на хостинге и настройки домена.

Сайт решает задачи:
- презентация услуг компании
- демонстрация собственных материалов
- предоставление прайса и контактов
- получение заявок от клиентов

## ⚙️ Что было сделано
- адаптивная верстка сайта
- разработка структуры страниц
- добавление новых блоков и страниц в процессе работы
- перенос сайта с конструктора на хостинг
- подключение домена
- настройка SSL (https)
- базовая SEO-оптимизация:
  - meta-теги (title, description)
  - robots.txt
  - sitemap.xml
  - подключение Яндекс Вебмастера
