# Polikrovsystems

Корпоративный сайт компании по устройству и ремонту мягкой кровли.

## Live Demo
https://yantilina.github.io/polikrovsystems/

## О проекте
Сайт разработан как презентационный ресурс компании:
- услуги по устройству и ремонту кровли
- собственные гидроизоляционные материалы
- контактная информация и прайс

## Технологии
- HTML5
- CSS3 (Flexbox, Grid)
- адаптивная верстка
- базовый JavaScript

## Структура
- главная страница
- блок о компании
- преимущества
- разработки и материалы
- услуги
- контакты

## Запуск
Открыть файл `index.html` в браузере  
или запустить локальный сервер.

## Статус
Проект в разработке. Добавляются страницы и адаптив.
